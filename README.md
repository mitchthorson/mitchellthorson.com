# mitchellthorson.com

This repository represents the source code for https://www.mitchellthorson.com, which is my personal website. It has information about me, my work etc.

It's built with SvelteKit.

## Developing

Install dependencies with `npm install` (or `pnpm install` or `yarn`), start a development server:

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

To create a production version of the site:

```bash
npm run build
```
